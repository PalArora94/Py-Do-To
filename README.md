# Py-Do-To
A command-line application to manage tasks.

This is a command-line application to manage tasks that has the following features:

1. Add tasks
2. View all the tasks in the list
3. Mark tasks as complete
4. Delete tasks
5. Download the list as a CSV file


## Workflow
- A text file will store the tasks. For each task, there will be a unique ID, a title, and a status, which will either be incomplete or complete.
- To manage all the features of our app, we will use a dictionary.
